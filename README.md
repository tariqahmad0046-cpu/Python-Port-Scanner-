# 🔥 Python Port Scanner

## 📌 Project Overview

This project is a simple Python-based port scanner used for basic network reconnaissance and cybersecurity learning.

It scans common ports on a target host and identifies whether ports are open or closed.

---

## 🎯 Features

- Fast port scanning
- Open port detection
- Closed port detection
- Simple socket programming
- Beginner-friendly cybersecurity project
